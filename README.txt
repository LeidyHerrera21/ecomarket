Demo EcoMarket

Archivos generados:
- index.html
- styles.css
- script.js

Instrucciones:
1. Descomprime el ZIP y abre index.html en un navegador con conexión a internet (se usan CDN para Bootstrap y qrcodejs).
2. Prueba agregar productos al carrito y pulsar 'Pagar' para generar el QR. Ingresa un número Yape/Plin y confirma el monto.

Nota: El QR creado contiene un texto legible (PAYMENT|phone:...|amount:...), algunas apps aceptan ese formato y otras requieren formatos oficiales. Si planeas usar esto en producción, coordina con los servicios de pago de Yape/Plin para el formato exacto o integra una pasarela certificada.
